# 📚 Research Paper Summarizer AI

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32+-red)

**Understand any research paper in 60 seconds.** Paste an arXiv link or upload a PDF — get a structured summary, key findings, and a plain-English explanation.

## Features
- 🔗 arXiv URL support (auto-fetches abstract + title)
- 📄 PDF upload and text extraction
- 📝 5-bullet executive summary
- 🔬 Methodology breakdown
- 💡 Key findings & contributions
- 🧑‍🏫 "Explain like I'm 18" plain English version
- ❓ Ask follow-up questions about the paper

## Run
```bash
pip install -r requirements.txt
GEMINI_API_KEY=your_key streamlit run app.py
```
